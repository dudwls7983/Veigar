# Veigar
LOL 클라이언트 로딩중 응답없음으로 인한 화면 멈춤 현상 발생
위 상황에서는 클라이언트 자체 종료(Alt+F4)와 작업관리자 가려짐 현상 발생 확인
이러한 이유로 이 현상을 해결 하기 위해 롤 클라이언트를 강제로 종료하는 프로그램 개발 착수


- Usage -
League of Legends 게임 내부에서 Alt + F4를 사용
